# VectorCMS Frontend

React + Vite + TypeScript + Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

The Vite dev server proxies `/api` requests to the backend at `http://localhost:8000`.

## Routes

| Path | Page |
|------|------|
| `/documents` | Drag-and-drop document upload and management |
| `/search` | Semantic search over indexed documents |

## Build

```bash
npm run build
```
